Este programa nos permite recibir numeros que el usuario ingresa y los va guardando
en diferentes estructuras dependiendo del tipo de Número, sea para pila numeros impares, para cola numeros pares
una lista enlazada para los numeros negativos.
Entonces siguiendo está logica, el usuario ingresa un numero, si es para se guarda en cola, si es impar se
guarda en la pila, si es negativo se guarda en la lista.
Esto gracias a la memoria dinamica, o sea que los nodos se crean con new y se eliminan con delete, sin usar arreglos
asi mismo el programa cuanta con un menu hecho con case, donde el usuario tiene 6 opciones para escoger.

La manera de ejecutarlo es sencilla, el usuario tiene que ingresar los numeros según el quiera, y de igual
 forma el puede realizar las busquedas que más le convengan.
